## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/samuelcasanova/adventjs2022.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Execute the unit tests that validate all the challenges
   ```sh
   npm test
   ```
